Setup steps:
  1. Download icon16.png, icon48.png, icon128.png → save to bookmark_ext/icons/
  2. Reload the extension at chrome://extensions (click ↺) — the flower icon appears in your toolbar
  3. Download bgm-square-logo.png → Square Dashboard → Account & Settings → Business Information → Logo
  4. Download bgm-square-banner.png → use as the cover image on your Square payment link page
  5. Download bgm-donation-image.png → use as the image on your Square donation link (1200×800, 3:2 ratio)
  6. Download bgm-screenshot-1280x800.png → Chrome Web Store listing → Screenshots (this is the one required image field)
  7. Download bgm-promo-small-440x280.png → Chrome Web Store listing → Small promo tile (optional)
  8. Download bgm-promo-marquee-1400x560.png → Chrome Web Store listing → Marquee promo tile (optional)
